AI-assisted crop recommendation

A practical appilaction for Indian farmers: combine soil nutrients (N, P, K), soil pH, and weather signals to rank suitable crops. Built for transparency— you can always enter values manually or use location-based weather when configured.

Start prediction

What this project does

Model

RandomForest classifier trained on cleaned soil–weather–label data with documented engineered features (nutrient totals, balance, climate interactions).

Weather

Fetch of current temperature and humidity via server

Insights

Ranked recommendations with crop detail pages for irrigation guidance. District yield charts load from an official APY CSV from dataset; the app does not invent yield values.

How it works

  1. Soil & weather — Enter N, P, K, pH, and weather (or use location / city fetch).
  2. Rankings — The model returns ranked crops with probabilities.
  3. Crop detail — Tap a crop for overview and irrigation notes (extension advice still applies).
